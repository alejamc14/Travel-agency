# TravelUp Colombia
## Description 
The project involves the design and construction of a static website for a travel agency using only HTML5 and CSS3. The objective is to present information about tourist destinations, services offered, and contact options in a visual and structured way, applying basic principles of web design, accessibility, and best practices in layout.

## Technologies used
HTML y CSS

## Installation or viewing instructions.
To view or edit the project, you only need:

+ A web browser (Chrome, Firefox, Edge, Safari, etc.)

+ (Optional) A code editor such as Visual Studio Code, Sublime Text, or Notepad++.

No server, database, or additional software is required since the project is built with HTML and CSS only.
